# Android Sync Template

This is a demonstration of how a basic sync app will work.  Refer to [fhconfig.properties](app/src/main/assets/fhconfig.properties) and [ListItemsFragment](app/src/main/java/redhat/com/syncsample/view/ListItemsFragment.java) for the delevant pieces of code and configuraiton.

## Prerequisites 
 * fh-android-sdk : 3.0.0-SNAPSHOT
 * Android Studio 1.4.0
 * Android SDK 22+

## Build instructions
 * Add fhconfig.properties
 * Attach running Android Device with API 16+ running
 * ./gradlew installDebug
